How Routing Works
1. Each target playlist is embedded using its name + description + custom notes blended with the centroid of its existing songs.
2. When a new track arrives (from liked songs or a source playlist), its vibe is compared to every target.
3. The closest match above the confidence threshold gets the track added directly. Others go to Inbox for manual review.
4. Target embeddings refresh every hour so they stay up to date as playlists grow.
